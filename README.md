1. **Source:** https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud?resource=download](https://www.kaggle.com/datasets/khusheekapoor/vehicle-insurance-fraud-detection/data
2. **Download:** Download the data manually from the link above

## Dataset
This project uses the Car Insurance Fraud dataset ("carclaims"), originally provided by Angoss Knowledge Seeker.

Key characteristics:

Total samples: 15,420

Legitimate claims: 14,497

Fraudulent claims: 923 (~6%)

Features: 32 total

25 categorical

6 ordinal

1 binary target variable (fraud vs not-fraud)
